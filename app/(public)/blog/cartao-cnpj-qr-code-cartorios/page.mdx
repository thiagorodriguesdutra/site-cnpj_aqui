# Cartão CNPJ com QR Code: por que os cartórios estão exigindo

Se você foi ao cartório para registrar um contrato, fazer uma alteração empresarial ou assinar uma escritura e ouviu que seu **Cartão CNPJ precisa ter QR Code**, saiba que isso já virou prática comum.

E não é implicância.

É uma mudança real na forma como os cartórios lidam com documentos empresariais.

---

## Por que os cartórios passaram a exigir QR Code

Cartórios têm uma função crítica: **dar fé pública**.  
Se um documento falso passa, o problema não é só de quem apresentou — o cartório pode responder judicialmente.

Nos últimos anos, aumentaram:
- PDFs adulterados
- Documentos empresariais desatualizados
- Uso indevido de dados de empresas ativas

Sem verificação digital, o tabelião não tem como saber se o documento é legítimo.

---

## O que mudou na prática

Antes:
- PDF impresso
- Conferência visual
- Confiança no papel

Hoje:
- Verificação digital
- Conferência online
- Validação em tempo real

Com o **QR Code**, o cartório escaneia o documento e acessa uma página de verificação que mostra os dados oficiais do CNPJ.  
Se algo não bater, o documento é recusado.

Simples assim.

---

## Como funciona o QR Code de autenticidade

O QR Code é um link único associado àquele documento específico.

Ele conecta o PDF a:
- Razão social
- CNPJ
- Situação cadastral
- Endereço
- Data e hora de emissão

Se alguém altera o PDF, o QR Code continua apontando para os dados originais.  
Na verificação, a fraude aparece imediatamente.

---

## É obrigatório em todos os cartórios?

Ainda não por lei.

Mas muitos cartórios já adotaram como **regra interna**, especialmente para:
- Contratos sociais
- Alterações contratuais
- Abertura de filiais
- Procurações empresariais
- Escrituras públicas
- Registros imobiliários em nome de empresa

Em grandes cidades, isso já é comum. A tendência é virar padrão nacional.
